There are 4 files : prac.h,prac.c.prac.l and prac.nc
prac.h has all the definations 
prac.c is a program to write output to output.txt after passing input file in lexical analyser
prac.l is lexical analyser program
prac.nc is input test program to test our analyser

To run the files write : "make clean" followed by : "make" iun terminal 
After that write : ./lexer prac.nc 
It will generate output file "output.txt" where tokens are present.
